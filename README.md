# Personal Finance / Expense Tracker

I created a console-based Personal Finance / Expense Tracker in **Python**. The project allows users to record, categorize, and track expenses, using Python programming, object-oriented design, data persistence, and basic data processing.

---

## Features
- Add expenses with **amount, category, description, and timestamp**  
- View all recorded expenses in a formatted table  
- Calculate **total expenses**  
- Persist data using **JSON storage**  
- Simple console interface for easy interaction  

---

## Technologies Used
- **Language:** Python 3  
- **Paradigm:** Object-Oriented Programming (OOP)  
- **Tools:** JSON for data storage, standard Python libraries  

---
